# DevOps USB Auditor
- Projeto de automação em Linux para auditoria de dispositivos conectados ao 
sistema.

## Objetivo
- Detectar automaticamente a conexão de dispositivos (USB, periféricos);
- Coletar informações relevantes e registrar eventos para 
rastreabilidade e observabilidade.

### Tecnologias
- Linux;
- Bash;
- Git;
- udev;(Futuro)
- docker;(Futuro)
