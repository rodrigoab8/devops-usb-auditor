# DevOps USB Auditor
Projeto de automação em Linux para auditoria de dispositivos conectador ao sistema.

## Objetivo
Detectar automaticamente a conexão de dispositivos (USB,controles, perifericos), coletar informações relevantes e registrar eventos para rastreabilidade e observabilidade.

### Tecnologias
-Linux
-Bash
-Git
-udev(futuro
-docker(futuro)
